Operador Hermano (+)
Al utilizar un operador +, se puede colocar diferentes elementos en el mismo nivel.
div+p+h1+h2
Operador Hijo (>)
Al utilizar un operador >, la siguiente linea sera incluida dentro de la linea anterior.
tr>th+td
Atributo Clase (.)
Al utilizar el simbolo ., permite asignar una clase al elemento a ingresar.
div.row
Operador ID (#)
Al utilizar el simbolo #, permite asignar un identificador al elemento a ingresar.
div#main
Texto ({})
Al utilizar llaves ({}), permite agregar texto al elemento a ingresar.
div{Hola}
Atributos ([])
Al utilizar corchetes ([]), permite agregar un atributo al elemento a ingresar.
tr[colspan="2"]
Multiplicación (*)
Al utilizar el simbolo *, permite agregar elementos un cierto numero de veces.
div*5
Numeración de Elementos ($)
Al utilizar el simbolo $, en caso de agregar un elementos varias veces, numerara la cantidad de elementos repetidos.
div*5{Numero $$}
